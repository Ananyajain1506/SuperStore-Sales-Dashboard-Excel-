📊 Superstore Sales Dashboard (Excel)


Overview


The Superstore Sales Dashboard is an interactive Excel-based business intelligence project developed to analyze sales performance, profitability, and order trends using the Sample Superstore dataset. The dashboard provides key business insights through dynamic visualizations, KPI cards, and interactive filters, enabling data-driven decision-making.


Dashboard Features

Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Total Orders


Interactive Visualizations
- Sales by Category
- Profit by Region
- Monthly Sales Trends


Dynamic Filters
- Region
- Category
- Segment
- Order Date Timeline


Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline Filters
- Data Visualization
- Dashboard Design


Business Insights
- Technology category generated the highest sales.
- The West region contributed the highest profit.
- Monthly sales analysis highlights seasonal trends and growth patterns.
- Interactive filters enable detailed analysis across regions, categories, and customer segments.


Skills Demonstrated
- Data Analysis
- Data Visualization
- Dashboard Development
- Business Intelligence Reporting
- Excel Automation
- KPI Tracking


Dataset
Sample Superstore Dataset


Project Objective
To transform raw sales data into meaningful business insights through an interactive Excel dashboard that supports performance monitoring and business decision-making.
